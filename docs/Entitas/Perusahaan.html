Perusahaan

Menu Perusahaan, digunakan untuk identifikasi Perusahaan bisnis unit.

Langkah-langkah mengakses menu Perusahaan​

  1. Buka halaman website dashboard dengan situs https://analytics.ecespro.id

       

  1. Setelah itu, jika user belum login maka silahkan login terlebih dahulu.

  2. Pilih modul Entitas lalu setelah itu pilih menu Perusahaan

Langkah-langkah menampilkan data list Perusahaan​

  1. Setelah kita berhasil mengakses menu Perusahaan maka sistem otomatis akan menampilkan data list Perusahaan.

Langkah-langkah menambahkan data Perusahaan​

Fungsi Tambah data berguna untuk membuat entri atau record baru dalam tabel database. 

  1. Pilih button "+ Tambah" pada bagian kanan atas tabel.
  2. Setelah berhasil memilih button "+ Tambah" maka sistem akan menampilkan modal pop-up yang telah tercantum informasi formulir yang akan di isi oleh user.
  3. User wajib mengisi formulir data perusahaan agar bisa lanjut ke langkah berikutnya. Terdapat 3 langkah formulir tambah Perusahaan, yakni :
    • Data Perusahaan
    • Data Pejabat
    • Data Pemegang Saham

  1. Langkah pertama melengkapi formulir data perusahaan, pilih Status.
  2. Pilih Subholding,
  3. Pilih Kode Perusahaan.
  4. Masukan Alias (opsional)
  5. Masukan Nama Perusahaan.
  6. Masukan Alamat Perusahaan.
  7. Masukan Kota asal Perusahaan.
  8. Masukan Provinsi asal Perusahaan.
  9. Masukan Kode Pos Perusahaan.
  10. Masukan nomor telepon Perusahaan.
  11. Masukan E-mail Perusahaan.
  12. Masukan situs website Perusahaan.
  13. Masukan PIC Perusahaan.
  14. Pilih Kategori Bisnis.
  15. Siapkan file berbentuk gambar lalu pilih file tersebut untuk upload atau user bisa melakukan drag dan drop file.
  16. Setelah itu, jika user sudah memastikan formulir data Perusahaan telah sesuai maka pilih button Simpan dan Lanjutkan untuk menyimpan data tersebut dan melanjutkan ke langkah berikutnya.
  17. Langkah kedua melengkapi formulir data pejabat, pilih button + Tambah pada bagian kanan atas tabel data pejabat.
  18. Setelah berhasil memilih button "+ Tambah" maka sistem akan menampilkan modal pop-up yang telah tercantum informasi formulir yang akan dilengkapi oleh user.

Langkah-langkah hapus data Perusahaan​

Fungsi Hapus data ini berguna untuk menghapus data di database yang sudah tidak diperlukan lagi. Ketika menggunakan fungsi ini, user mengakses detail entri dan kemudian menginstruksikan sistem untuk menghapusnya dari database.

  1. Pilih baris data dengan meng-klik icon titik tiga pada bagian kiri baris data.
  2. Pilih dan klik delete.
  3. Setelah di klik, sistem akan menampilkan pop-up konfirmasi yang tercantum keterangan bahwa data yang telah terhapus tidak akan bisa dipulihkan kembali.
  4. Pilih dan klik Ok untuk menghapus data dan pilih Cancel untuk membatalkan.
  5. Setelah berhasil menghapus data, maka sistem akan menampilkan pesan Record Deleted pada bagian kanan atas tabel.

Langkah-langkah preview data Perusahaan​

Fungsi Preview adalah untuk melakukan pratinjau data yang telah di entry oleh user.

  1. Pilih baris data dengan meng-klik icon titik tiga pada bagian kiri baris data.
  2. Pilih dan Klik Preview
  3. Setelah di klik, maka sistem akan menampilkan pop-up yang telah tercantumkan informasi data pada setiap baris data yang dipilih.

Langkah-langkah menggunakan fitur-fitur pada tabel Perusahaan​

Fitur adalah fitur khusus yang disertakan dalam alat. fitur yang telah tersedia pada Perusahaan antara lain, yakni :

  1. Fitur filter select Subholding User dapat memfilter banyak data Subholding sesuai dengan kriteria yang telah ditentukan pada Subholding.
  2. Fitur pencarian data Fungsi pencarian data memungkinkan user memfilter data sesuai dengan kriteria.
  3. Fitur show/Hide tabel Fungsi show/hide tabel adalah untuk menampilkan atau menyembunyikan field pada tabel.
  4. Fitur ukuran tabel. Berfungsi untuk mengatur ukuran tabel.
  5. Fitur export PDF Fungsi dari export PDF yaitu untuk mengeluarkan dan menyimpan data supaya dapat di Import kedalam file berbentuk PDF.
  6. Fitur export Excel Fungsi dari export Excel yaitu untuk mengeluarkan dan menyimpan data supaya dapat di Import kedalam file berbentuk Xls.
