Fundamental

Langkah-langkah mengakses menu Fundamental​

  1. Buka halaman website dashboard dengan situs https://analytics.ecespro.id

       

  1. Setelah itu, jika user belum login maka silahkan login terlebih dahulu.

  2. Pilih modul Dashboard lalu setelah itu pilih menu Fundamental.

Langkah-langkah menampilkan data list Fundamental​

Fundamental​

  1. Setelah kita berhasil mengakses menu Fundamental maka sistem otomatis akan menampilkan data list Fundamental.
  2. Tab Fundamental mempunyai filter data yaitu :
  • Filter bulan, quarter, tahun.
  • Filter Grup Entitas.
  • Filter Subholding 1.
  • Filter Subholding 2.
  • Filter Perusahaan.
  • Filter Bisnis Unit.
  1. Klik Terapkan untuk menampilkan data pada tabel dan chart.

Rasio Likuiditas​

  1. Setelah kita berhasil mengakses menu Fundamental maka sistem otomatis akan menampilkan data list Fundamental lalu pilih tab rasio likuiditas.
  2. Tab Fundamental mempunyai filter data yaitu :
  • Filter bulan, quarter, tahun.
  • Filter Grup Entitas.
  • Filter Subholding 1.
  • Filter Subholding 2.
  • Filter Perusahaan.
  • Filter Bisnis Unit.
  1. Klik Terapkan untuk menampilkan data pada tabel dan chart.

Rasio Profitabilitas​

  1. Setelah kita berhasil mengakses menu Fundamental maka sistem otomatis akan menampilkan data list Fundamental lalu pilih tab rasio profitabilitas.
  2. Tab Fundamental mempunyai filter data yaitu :
  • Filter bulan, quarter, tahun.
  • Filter Grup Entitas.
  • Filter Subholding 1.
  • Filter Subholding 2.
  • Filter Perusahaan.
  • Filter Bisnis Unit.
  1. Klik Terapkan untuk menampilkan data pada tabel dan chart.
